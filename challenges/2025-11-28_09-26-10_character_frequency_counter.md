# Character Frequency Counter

## Difficulty
Easy

## Description
Write a C++ program that takes a string and a single character as input. The program should then count and print the total number of times the given character appears in the string. The comparison should be case-sensitive.

## Constraints
*   The input string will contain only ASCII characters.
*   The length of the string will be between 1 and 100 characters, inclusive.
*   The input character will be a single ASCII character.

## Test Cases
### Test Case 1
#### Input
```
hello world
l
```
#### Output
```
3
```

### Test Case 2
#### Input
```
programming
x
```
#### Output
```
0
```

### Test Case 3
#### Input
```
Banana
a
```
#### Output
```
3
```

### Test Case 4
#### Input
```
Mississippi
s
```
#### Output
```
4
```

### Test Case 5
#### Input
```

```

10
